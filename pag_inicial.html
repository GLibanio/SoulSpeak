QUEM SOMOS

Desenvolvido para aqueles que desejam ter um espaço aberto para obter conhecimento sobre saúde mental, o SoulSpeak é um site que visa democratizar o acesso a saúde mental.

Aqui você encontra conteúdo sobre diversos assuntos, como ansiedade, depressão, luto, entre outros.

Conta ainda, com um espaço para registro de sentimentos e desabafos, as vezes tudo o que você precisa é colocar para fora, mas não sabe como, então esse espaço é para você!
